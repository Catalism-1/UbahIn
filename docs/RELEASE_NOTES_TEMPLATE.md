# Ubahin v0.1.1-beta

## Status

Ini adalah versi beta untuk pengujian publik. Gunakan untuk mencoba fitur dasar dan laporkan masalah melalui GitHub Issues.

## Fitur yang tersedia

* Konversi PDF ke JPG secara lokal/offline
* Pilihan kualitas Standard, Tinggi, dan Sangat Tinggi
* Dukungan antrean hingga 50 PDF
* Progress proses dan pembatalan
* Opsi ZIP hasil
* Installer Windows dan versi portable

## Cara instalasi

1. Unduh `Ubahin_Setup.exe`.
2. Jalankan file installer.
3. Klik Install.
4. Setelah selesai, klik Finish.
5. Buka Ubahin dari shortcut Desktop atau Start Menu.

## Catatan

* Saat ini fitur utama yang siap digunakan adalah PDF ke JPG.
* Beberapa fitur dan desain UI masih dalam tahap pengembangan.
* Semua file diproses secara lokal di perangkat pengguna.

## Laporkan masalah

Buat Issue baru dengan:

* versi Ubahin
* versi Windows
* langkah yang dilakukan
* screenshot error bila ada
* file log jika diperlukan, tanpa mengunggah dokumen pribadi
